Collapse-UI
Collapse-UI is essentially a Bedrock Edition's Custom UI to make it looks like console as possible.

It was Arcdustry-UI, but rewritten from scratch aswell same goes to buttons and couple of things.

This UI is basically a more of console-alike UI with ease of access to mobile players. It's also customizable via "Collapse-UI/ui/_global_varibles.json" and few couple of tweaks on the way layout looks.

This project is a replacement to my previous UI project and will be in maintained in forseenable future.
