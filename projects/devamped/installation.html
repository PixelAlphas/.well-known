Installation

To install Devamped, You can grab the download in the download tab, you basically install it like every texture packs. if you didn't know anyhow, here's an instructions.

  1. Download Devamped with .mcpack format, if you've received the .zip version. you can rename it to .mcpack
  2. Import the .mcpack to Minecraft by clicking/tapping and "Open with". (For android users, You can use Zarchiver from playstore, press the arrow next to "Open" )
  3. The game will launch and will automatically install Devamped for you.
  4. Goto — Game settings > Global Resource > Apply/Activate "Devamped"
  5. You are done! — To revert, just deactivate like any texture packs!

For Java, well. you definitely know how to.
